Project Description:
This is a frontend-only web application designed to support farmers and promote rural development. The website provides practical tools, guidance, and resources to improve agricultural practices, crop planning, and community engagement. It is built entirely using HTML, CSS, and JavaScript, making it lightweight, responsive, and easy to use.

Key Features

Interactive AI Chatbot: A floating virtual assistant that answers questions about crops, irrigation, pests, weather, and government schemes. Includes images and quick suggestion buttons for a more visual experience.

Weather Advisory: Displays local weather, rainfall, and humidity information to help farmers plan irrigation and crop care.

Success Stories: Highlights inspiring stories of farmers to motivate and educate the community.

Tips and Guidelines: Practical advice on modern farming techniques, pest management, and crop planning.

Government Schemes & Subsidies: Information on relevant schemes to help farmers access financial support.

Responsive Design: Works on desktops, tablets, and mobile devices for easy access in rural areas.
